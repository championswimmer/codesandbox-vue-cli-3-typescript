# CodeSandbox Vue CLI v3 Template with TypeScript Support

## Running on CodeSandbox

Open this link -
https://codesandbox.io/s/github/championswimmer/codesandbox-vue-cli-3-typescript


## Running Locally

### Install Dependencies

```shell
yarn
# or npm install
```

### Serve Locally (Port: 8080)

```shell
yarn serve
# or npm run serve
```
